A full-stack MERN web app to view and predict real-time and historical air quality data for Delhi.

## Features
- Real-time AQI and pollutant data by station
- Interactive map and pollutant cards
- Date/hour-based search and 24-hour/daily trends
- ML-powered AQI predictions

## Tech Stack
- **Frontend**: React.js, Tailwind, Leaflet, Recharts
- **Backend**: Node.js, Express, MongoDB
- **ML API**: Python + Flask

## Setup
- `cd client && npm install && npm run dev`
- `cd server && npm install && node index.js`
